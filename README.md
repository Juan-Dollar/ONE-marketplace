# ONE - Construction & Skilled Labor Marketplace

A comprehensive platform connecting skilled laborers with clients for construction and home improvement projects.

## Features

- ✅ User Authentication (Workers & Clients)
- ✅ Worker Profiles & Portfolios
- ✅ Job Listings & Applications
- ✅ Real-time Messaging
- ✅ Secure Payment Processing (Stripe)
- ✅ Rating & Reviews System
- ✅ Subscription Plans
- ✅ Transaction-Based Revenue Model
- ✅ Job Posting Fees
- ✅ Background Verification

## Tech Stack

- **Frontend**: React.js, Tailwind CSS, Axios
- **Backend**: Node.js, Express.js
- **Database**: MongoDB Atlas (Free)
- **Real-time**: Socket.io
- **Payments**: Stripe
- **File Storage**: Cloudinary
- **Hosting**: Render (Backend), Vercel (Frontend)

## Quick Start

### Prerequisites
- Node.js (v14+)
- npm or yarn
- Git

### Installation

```bash
# Clone repository
git clone https://github.com/Juan-Dollar/ONE-marketplace.git
cd ONE-marketplace

# Install dependencies
npm install

# Create .env file
cp .env.example .env

# Start development
npm run dev
